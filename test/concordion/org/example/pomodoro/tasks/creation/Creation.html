Task Creation

There is a Create Task form you can use to store a new task.

Given the following values for a task:

Summary Details Status Deadline Day Deadline Month Deadline Year Time Spent
Write concordion specs Runnable specifications for the new project Open 1 1 2025 0

When a user fills in the Create Task form with the previous values the task will be saved successfully and the list of tasks contains one with "Write concordion specss" as summary.
